svn-revision-number:revision

Full name:

com.google.code.maven-svn-revision-number-plugin:maven-svn-revision-number-plugin:1.2:revision

Description:

Retrieves the revision number and the status of the Subversion working copy directory.

Attributes:

  • Requires a Maven 2.0 project to be executed.
  • Binds by default to the lifecycle phase: initialize.

Required Parameters

Name Type Since Description
workingCopyDirectory File - The Subversion working copy directory. The plugin will evaluate the aggregated status and revision number of this directory and its contents.
Default value is: ${basedir}.

Optional Parameters

Name Type Since Description
reportIgnored boolean - Whether to collect the status information on items that were set to be ignored.
Default value is: false.
reportMixedRevisions boolean - Whether to report the mixed revisions information. If set to false then only the maximum revision number will be reported.
Default value is: true.
reportOutOfDate boolean - Whether to check the remote repository and report if the local items are out-of-date.
Default value is: false.
reportStatus boolean - Whether to report the status information. If set to false then only the revision number will be reported.
Default value is: true.
reportUnversioned boolean - Whether to collect the status information on items that are not under version control.
Default value is: true.
verbose boolean - Provides detailed messages while this goal is running.
Default value is: false.
